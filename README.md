# Página Coffeeholic (Viciados por café!)

### Concepção:

- Criar a aplicação por completo deste UI á codificação. O estilo da página é de uma loja de cafés onde terá categorização, carrossel é as possiveis páginas. Utilizei para criar a UI (FIGMA) é na parte da coficiação (NEXT.JS + TailwindCSS)
- Objetivo era criar o Layout (FIGMA)
- Codar o projeto de acordo as especificações do Layout.

### Resolução:
- Inicialmente, conduzi uma pesquisa para identificar o layout desejado e ter uma noção da posição de cada container. Para me inspirar, utilizei o site Dribbble em busca de referências. Em seguida, após fazer minhas decisões, pesquisei sobre a paleta de cores que melhor se adequaria ao projeto. Com o conceito definido, levei-o para o FIGMA e me dediquei a aplicar alguns recursos avançados, como o Auto Layout, Frame e Groups, com o objetivo de aprimorar ao máximo o design. [VISUALIZE O PROJETO FIGMA](https://www.figma.com/file/r3fpgpX0NnPgh7YSRgWlzX/Coffee-store?type=design&node-id=0%3A1&mode=design&t=yQEBNvkavdaIxZ8R-1)
- Parte da codificação: Utilizei Next.js 13 + TailwindCSS. Codificação foi de acordo ao layout no figma, utilizei o EsLint para facilitar a identação do código. Utilizei algumas bibliotecas novas nesse projeto que me permitiram aprimorar meu conhecimentos, como KEEN-SLIDER (Trabalhar com Carrosel de forma simplificada!) 
 -Tanto o Header + Sidebar -> São fixos!

 ## :rocket: Techs
 
 * Next.js + TypeScript
 * TailwindCSS
 * Keen-slider
 * Eslint
 * Lucide-react
